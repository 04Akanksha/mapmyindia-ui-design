# MapmyIndia Webpage UI Design

This project is a UI/UX design replica of the MapmyIndia homepage.
The design has been created using **draw.io** for learning and practice purposes.

##  Project Overview
- Platform: draw.io
- Type: UI/UX Webpage Design
- Inspired by: MapmyIndia official website
- Purpose: Educational & practice project

##  Tools Used
- draw.io (for UI design)
- GitHub (for version control and hosting)
- Youtube (for learning)

##  Features of Design
- Clean and modern UI layout
- Navigation bar structure
- Map section layout
- Service & feature sections
- Footer layout
